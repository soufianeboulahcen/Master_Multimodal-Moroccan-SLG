## JSON keypoint format

Each file follows the OpenPose 1.3 schema:

```json
{
  "version": 1.3,
  "people": [{
    "person_id": [-1],
    "pose_keypoints_2d":       [...],
    "face_keypoints_2d":       [...],
    "hand_left_keypoints_2d":  [...],
    "hand_right_keypoints_2d": [...]
  }]
}
```

Coordinates are in pixels (1280x720). Confidence is 1.0 for all generated keypoints.

## Visual effects applied

- Glow / bloom: additive Gaussian blur blended over skeleton lines
- Cinematic bars: 8% letterbox at top and bottom
- Camera drift: slow sinusoidal translation (+-6 px horizontal, +-4 px vertical)
- Animated background: dark gradient with vignette and slow colour pulse (overlay version)
