# outputs/

Generated artefacts from the Multimodal Moroccan Sign Language Generation pipeline.

## videos/
Per-clip MP4 outputs in 7 render styles:

| Folder | Description |
|---|---|
| `overlay/` | Skeleton drawn on original frame |
| `skeleton/` | Skeleton on pure black background |
| `neon/` | Glowing neon skeleton (Gaussian blur glow) |
| `heatmap/` | Confidence heatmap blended over original |
| `slowmo/` | 0.5× speed (frame duplication) |
| `studio/` | Skeleton on dark vignette gradient background |
| `mosaic/` | 3×2 tile of all 6 variants side-by-side |

## openpose_overlay/
Original MoSL clips with OpenPose-style skeleton overlaid.

## openpose_json/
Per-frame OpenPose-compatible JSON keypoints:
- `pose_keypoints_2d` — 18 COCO body joints × (x, y, confidence)
- `hand_left_keypoints_2d` — 21 hand joints
- `hand_right_keypoints_2d` — 21 hand joints
- `face_keypoints_2d` — 478 face mesh points

## figures/
Plots and animations generated by `final_project.ipynb`.
